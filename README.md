📚 RAG 入门学习项目（基于 LangChain）
> 本项目旨在帮助初学者从零开始学习和实践 RAG（Retrieval-Augmented Generation） 技术，并使用 LangChain 框架 构建一个完整的检索增强生成系统。

🛠️ 技术栈
 - Python 3.10+ 
 - LangChain
 - [OpenAI API / DashScope / HuggingFace]（任选其一作为 LLM）
 - Markdown/PDF/Text 文档解析工具
 - Embedding
